# CLI 기초



## CLI와 GUI

* CLI : Command Line Interface
* GUI: Graphic User Interface



## 경로

* 절대 경로 : 시작지점부터의 경로 (최상위부터..  root)
* 루트(디렉토리)  -> /
* 홈디렉토리 : 사용자 디렉토리 -> ~
* 상대경로: 현재 디렉토리부터의 경로



## 터미널 명령어

### cd

*  Change Directory

### mkdir

* Make Directory

### ch~

* 홈 디렉토리

### ch /

* 루트

### ch ..

* 상위

### ls

* 현재 작업중인 폴더에 포함되어있는 파일/폴더들의 목록을 출력
  * 숨김파일 포함 출력: -a
  * -l : 파일에 대한 추가적인정보, 수정날짜, 권한, 용량
