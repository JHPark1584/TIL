## git branch

* git branch

  * 브랜치 목록 확인

* git branch 브랜치이름

  * 새로운 브랜치 생성

* git branch -d 브랜치이름

  * 특정 브랜치 삭제 (병합된 브랜치만 삭제)

* git branch -D 브랜치이름

  * 강제 삭제

* git switch 브랜치 이름

  * 다른 브랜치로 이동

*  git switch -c 브랜치 이름

  * 브랜치를 새로 생성과 동시에 이동
