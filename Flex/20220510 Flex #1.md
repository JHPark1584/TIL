# 20220510 Flex #1



## Flex

* flex는 내용물의 width 만큼만 가져간다
* 칼럼의 높이가 자동으로 맞는다.



## Inline-flex

* container가 items의 내용물 만큼만 가져간다.



## 배치 방향 설정 flex-direction

* flex-direction
  * row
  * column
  * row-reverse
  * column-reverse



## 줄넘김 처리 설정 flex-wrap

* nowrap
  * container width넘어가면 삐져나감
* wrap
  * 줄바꿈이 일어남
* wrap-reverse
  * 줄바꿈이 위로
